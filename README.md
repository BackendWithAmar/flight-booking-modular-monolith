# Flight Booking Platform (Modular Monolith)

Backend system for a scalable travel booking platform.

## Tech Stack

Java  
Spring Boot  
Spring Security  
Spring Data JPA  
PostgreSQL

## Architecture

Modular Monolith with domain-based structure.

Modules:

- user
- search
- booking
- payment
- rewards
- notification

## Current Features

- User registration
- Password hashing
- DTO-based API design
- Global exception handling

## Next Steps

- JWT authentication
- Flight search
- Booking service
- Microservices extraction
