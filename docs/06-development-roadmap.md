# 06 – Development Roadmap

The project follows an Agile sprint-based development strategy.

---

## Sprint 0 – Project Setup

* Spring Boot project initialization
* Modular project structure
* PostgreSQL configuration
* GitHub repository setup

---

## Sprint 1 – Authentication System

Completed:

* User entity
* User repository
* DTO architecture
* Service layer
* Controller layer
* Password hashing
* Login API
* JWT token generation
* JWT authentication filter
* Protected endpoints

---

## Sprint 2 – Flight Search Module

Planned features:

* Flight entity
* Flight repository
* Flight search API
* Filtering
* Pagination

---

## Sprint 3 – Booking Service

Planned features:

* Booking creation
* Seat reservation
* Booking history

---

## Sprint 4 – Payment Service

Planned features:

* Payment processing
* Payment status tracking
* Retry handling

---

## Sprint 5 – Event-Driven Architecture

Planned features:

* Kafka event streaming
* Booking events
* Payment events
* Notification events

---

## Sprint 6 – Infrastructure and Deployment

Planned features:

* Docker containerization
* Service networking
* Local deployment environment

---

## Future Enhancements

* Frontend integration
* AI-powered recommendations
* Distributed tracing
* Observability dashboards
